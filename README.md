# Forms Tools

<img src="https://github.com/PHP-Sources/.github/raw/main/logos/forms-tools.webp" width="48" height="48" alt="Forms Tools">

**Forms Tools 3.1.1** – L’outil open source puissant et flexible pour créer et gérer des formulaires avancés.

### Points forts
- Création de formulaires complexes sans coder
- Gestion complète des soumissions et des données
- Interface d’administration intuitive
- Support des champs conditionnels et des vues personnalisées
- Export des données (Excel, CSV, etc.)
- Très utilisé pour les enquêtes, inscriptions, demandes et collectes de données
